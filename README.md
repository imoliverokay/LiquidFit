# LiquidFit Assets

Asset hosting for **LiquidFit**, an outfit studio script for Cherax, by oli.

The script downloads these automatically on first launch:

- `assets/essentials.zip` - item name data, category icons, logo
- `assets/thumbs_part1..5.zip` - the optional full preview image pack (~415 MB extracted, 27,848 images)

Preview images sourced from the public ShortByte/GTA5-Cloth-Assets and SheiylaDev/gta5-clothing-images archives, resized to 128px.
